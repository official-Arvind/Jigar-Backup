## ⚡ Overview

Jigar Backup runs natively on your device. It utilizes Root permissions to bypass standard Android scoped storage limitations, instantly indexing your installed packages using direct Linux system commands.

The built-in **C++ Titan Engine** handles all the heavy lifting—compressing gigabytes of data using ZSTD before safely exporting them as archives locally or streaming them directly to your personal Cloud NAS.
