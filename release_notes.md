# 🚀 Jigar Backup V1.0

The highly anticipated V1.0 release is finally here! This version brings the full power of the Titan Backup Engine directly to your Android device, delivering unmatched backup performance.

## ⚡ What Is Included

- 🌳 **Root-level Extraction** (Magisk, KernelSU, APatch)
- 😎 **Native Zstandard Compression** for 100% data integrity
- ☁️ **Network Sync** (Direct SMB / WebDAV integration)
- 🌀 **Multi-user & Work Profile Support**
- ⚡ **Lightning Fast Speeds** saturating internal storage limits
- ☀️ **Telemetry Free & Open Source**

## 🛠️ Installation

Simply download the attached `.apk` below and install it on your rooted Android device.

> [!IMPORTANT]  
> **Root Access Required:** Jigar Backup requires Root permissions (Magisk/KernelSU) to bypass scoped storage constraints and index packages at the Linux system level.
